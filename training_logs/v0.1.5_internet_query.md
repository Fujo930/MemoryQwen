## Key Design Decisions

- web.enabled defaults to false
- chat does not use web unless --web flag is set
- WebNeedDetector prevents web on local project questions
- [S] local / [W] web citation separation
- Only web ingest writes to memory
- No crawler, no recursive browsing, no auto-ingest

## Security Properties

- URL safety: file://, localhost, private IPs blocked
- Prompt injection: 8 patterns detected, logged, ignored
- Max bytes/chars enforced per fetch
- Sensitive query params redacted in logs
